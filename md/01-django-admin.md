## 장고 프로젝트 생성 
~~~
$ django-admin.py startproject mysite
~~~

- 프로젝트 명을 source로 변경 
~~~
/source/
    /mysite/
        __init__.py
        settings.py
        urls.py
        wsgi.py
    manage.py
~~~    

<br>
- 장고 서버 실행 
~~~
$ python manage.py runserver
~~~

- admin
    - http://localhost:8000/admin/
